## Set This Theme

1 - After Installing the theme,Open Command Pallet(`Ctrl + Shift + P`)

2 - Search for 'Preferences: Color Theme'
> `Keyboard Shortcut : Ctrl + K Ctrl + T`

3 - Search for 'Red One' and select the theme
